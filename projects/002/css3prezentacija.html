Ostala obilježja CSS3 - media upiti

  • Preciznije označavanje stranica kaskadnog stila
  • Primjerice, moguće je definirati jednu stranicu kaskadnog stila za prikaz stranice na monitoru, a drugu za ispis na papiru
@media screen and (max-width: 600px) {
    #media-query {
        background: Blue;
    }
}
@media screen and (min-width: 900px) {
    #media-query {
        background: Red;
    }
}
@media screen and (min-width: 600px) and (max-width: 900px) {
    #media-query {
        background: Green;
    }
}
